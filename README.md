# JTB 東京着地型 探究プログラム ポータル デモ

GitHub Pages 公開用の静的サイトです。

## 公開方法
1. このZIPを解凍
2. GitHubリポジトリ直下に `index.html`, `style.css`, `app.js`, `.nojekyll`, `assets/` を配置
3. Settings > Pages > Branch を `main` / `/root` に設定
4. 公開URLで確認

## 申込導線について
- デモでは `app.js` の `JTB_APPLICATION_URL` に仮URLを設定しています。
- 本番ではJTB指定サイトのURLに差し替えてください。
- 見積・選択条件はJSONとして生成され、JTB側送信用データとして確認・コピーできます。

## 主な機能
- 企業プログラム検索・カテゴリ絞り込み
- 期間・人数・学校情報による概算見積
- JTB送信用データ生成
- 展示・販売コーナー検索
- 三越伊勢丹デモプログラム
- リアライズ社 ビジネス甲子園導線
